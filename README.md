# Flipkart Automation Testing

## Project Overview
This project automates the core functionalities of the Flipkart website using Selenium WebDriver and Java. It follows the Page Object Model (POM) design pattern to create reusable and maintainable test scripts.

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Cucumber BDD
- Page Object Model (POM)

## Features Automated
- User Login
- Product Search
- Product Selection
- Add to Cart
- Test Execution with TestNG
- HTML/Extent Report Generation

## Project Structure
- src/
- pom.xml
- testng.xml
